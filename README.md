# Irreversible Dynamics in Financial Markets

This repository contains the data, code, and selected empirical outputs behind the study of *irreversible dynamics and structural responses of financial markets under stress*.

The project investigates whether scaling-law parameters derived from long historical financial time series behave as dynamic state variables encoding market stress and structural regime shifts.

The repository is structured to ensure clarity, reproducibility, and methodological rigor.

---

## 📌 Motivation

Financial markets exhibit complex statistical properties, including:

- Heavy-tailed return distributions  
- Volatility clustering  
- Long-range dependence  
- Multi-scale dynamics  

Traditional volatility models treat parameters as static calibration constants.  
This project explores whether scaling parameters themselves evolve over time and systematically respond to financial stress.

Understanding these dynamics provides insight into:

- Structural fragility  
- Regime shifts  
- Stress propagation mechanisms  
- Market adaptation versus structural reset

## 📂 data/ — Financial Inputs & Derived Series

The `data/` directory separates raw financial inputs from processed estimation outputs.

### raw/
- `PRICE_DEF.txt` — Historical daily DJIA closing prices (starting 1896).

### processed/
- `rigid_windows_results.csv` — Rolling estimation outputs.
- `rigid_windows_results_125.csv` — Rolling estimation using 125-day windows.

Processing steps (returns, rolling windows, alignment) are fully reproducible via `/src`.

## 📓 notebooks/ — Interactive Analysis

The `notebooks/` directory contains annotated Jupyter notebooks that:

These notebooks provide transparency and guided reproducibility.

## 💻 src/ — Computational Engine

The `src/` directory contains the core research implementation.

### 1. Data Transformations
- Log-return computation  
- Rolling window construction  
- Realized volatility calculation  

### 2. Scaling Parameter Estimation
- β — Volatility autocorrelation decay exponent  
- Dₑ — Effective anomalous diffusion exponent  
- Spline inversion of simulated calibration curves  

### 3. Monte Carlo Simulation
- Truncated Lévy autoregressive simulation  
- Generation of theoretical β(D) mapping  

### 4. Stress Dependence Testing
- Toda–Yamamoto Granger causality  
- Quantile predictability  
- Local Projections (Jordà framework)  
- Distance correlation  
- Wavelet coherence  

### 5. Visualization
- Automated generation of all figures in `results/`  

The code emphasizes modularity, transparency, and reproducibility, and can be extended to alternative indices or stress proxies.

## 🔄 Reproducibility Workflow

The full empirical pipeline follows:

Raw DJIA Prices  
→ Log Returns  
→ Rolling Window Estimation  
→ Stress Alignment  
→ Statistical Testing  
→ Visualization  

No manual post-processing is applied.
