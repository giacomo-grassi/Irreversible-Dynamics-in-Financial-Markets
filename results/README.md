# results/ — Selected Empirical Outputs

This directory contains representative examples of the empirical outputs generated by the rolling estimation and stress-dependence analysis pipeline.

The figures included here illustrate key aspects of the study.  
They do not exhaust the full set of results produced during the research process.

All outputs are generated programmatically via the modules implemented in `/src`.

---

## 📊 Figures (Selected Examples)

The following plots are illustrative outputs from the estimation framework:

### timeseries_overview.png
Rolling time-series of selected scaling parameters:

- β — Volatility autocorrelation decay exponent  
- Dₑ — Effective anomalous diffusion exponent  
- corr(β, Dₑ) — Rolling structural coupling  

This figure highlights the long-run structural evolution of scaling dynamics over the historical sample.

---

### stress_target_scatter.png
Cross-sectional relationship between:

- Financial stress proxy  
- Estimated scaling parameters  

Used to assess nonlinearity and state-dependence in the stress–parameter mapping.

---

### local_projections_irf.png
Impulse response functions (IRFs) obtained via the local projections (Jordà) framework.

Shows the dynamic response of selected scaling parameters to a unit financial stress shock, distinguishing short-term adaptation from slower structural adjustment mechanisms.

---

### wavelet_coherence.png
Time-frequency representation of dependence between financial stress and scaling parameters.

Highlights multi-scale coupling and regime-dependent coherence.

## 📈 Structured Output

### results.json (if present)

Contains machine-readable summary statistics and test results, such as:

- Estimated coefficients  
- Confidence intervals  
- Significance levels  
- Rolling summary metrics  

This file enables reproducibility and downstream analysis without re-running the full computational pipeline.

## 🔄 Reproducibility

All figures and structured outputs follow the pipeline:

Raw DJIA Prices  
→ Log Returns  
→ Rolling Parameter Estimation  
→ Stress Alignment  
→ Statistical Testing  
→ Visualization  

No manual post-processing was applied.
